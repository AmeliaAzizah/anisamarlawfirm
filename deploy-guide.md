# 🚀 Panduan Deploy Website Law Firm ke Netlify (GRATIS)

## Kenapa Netlify?
✅ 100% Gratis untuk static website  
✅ HTTPS otomatis  
✅ URL custom gratis (misal: pratama-law.netlify.app)  
✅ Drag & drop — tidak perlu coding  
✅ Bisa connect domain sendiri nantinya  

---

## Cara Deploy (5 Menit, Tanpa Akun GitHub)

### Langkah 1 — Buka Netlify Drop
👉 Buka: **https://app.netlify.com/drop**

### Langkah 2 — Upload Folder
- Extract/unzip file `lawfirm-website.zip`
- **Drag & drop folder `lawfirm`** ke halaman Netlify Drop
- Tunggu upload selesai (~30 detik)

### Langkah 3 — Selesai! 🎉
Netlify otomatis memberi URL seperti:
`https://amazing-tesla-123456.netlify.app`

---

## Cara Ganti URL Netlify (Opsional)
1. Daftar/login ke netlify.com
2. Klik site kamu → Site settings
3. General → Site name → ubah jadi `pratama-law`
4. URL jadi: `https://pratama-law.netlify.app`

---

## Cara Ganti Nama Law Firm
Cari dan ganti teks ini di semua file HTML:
- `Pratama & Associates` → nama firm kamu
- `pratama-law.com` → email kamu
- `+62 21 5555 0000` → nomor telepon kamu
- `Jl. Jend. Sudirman No. 123, Jakarta Selatan` → alamat kamu
